# 写成一个自定义函数，参数为hold_period, target_period, n=10,rank=top or bottom,表示holding_period是买入前十还是后十
# 函数返回target_period回报率之和

def buy_fund(holding_period,target_period,n=10,rank='top'):
    if rank=='bottom':
        ranking_method=True
    else:
        ranking_method=False
        
    df2=df[['Code',str(holding_period),str(target_period)]].sort_values(by=str(holding_period),ascending=ranking_method)[:n].reset_index(drop=True)
    ret_target_period=df2[str(target_period)].sum()
    
    return round(ret_target_period,4)

4. 业绩归因¶
    

  • 市场存在机制转换（Regime Switching）问题
    

    波动率的机制转换指的是波动率分为“持续型，consistent”和“疯狂型，mild”，也就是波动率自身呈现均值回复现象，高低波动率相互交错产生
    

    假设：基金经理风格无变化（系统性交易）
    

    “趋势-区间二分法”
    

    暗示

5. 总结¶
    

  • 量化研究方法是客观、准确的，是科学研究的途径
    • 



    

  • 分组研究方法
    

    seminal paper：Jagadeesh and Titman (1993)
    

    http://www.business.unr.edu/faculty/liuc/files/BADM742/Jegadeesh_Titman_1993.pdf
    

    分组研究方法步骤：
    

      

    1. 陈述、论证假设
      2. 

    2. 收据数据，根据假设将数据分为定序的5组
      3. 

    3. 分别对定序组进行假设统计，并比较“观察期”和“持有期”的绩效
      4. 

    4. 可根据定序组创建对比投资组合，例如动量研究时，将所有股票的观察期动量进行排序，前10%和后10%的股票分别组成投资组合
      5. 

    5. 对比定序组的长期绩效差异
      6. 

    

    • 



    

  • 研究实例：
    

    1.基于财务公告的波动率组合策略
    


      分组方法：GICS行业分类

  交易工具：+/- straddle (+ call +put / -call -put）

  回测时间段：2007~2020 美股期权

  回测结果：IT类别股票适合做多跨式期权也适合做空跨式期权
    

    • 




2.基于投资者异质性的动量策略

    分组方法：Hofstede个人主义指数，将投资者分为五组，个人主义指数递增，集体主义者......个人主义者

    交易工具：美股

    回测时间段：2000~2018 美股日线图

    回测结果：个人主义指数强的股票更适合动量交易
